# 🌙 Growlendário 2026

![Versão](https://img.shields.io/badge/version-1.0-66bb6a)
![Licença](https://img.shields.io/badge/license-MIT-05080a)

**Growlendário** é um calendário astronômico interativo desenvolvido para cultivadores que buscam otimizar seus resultados utilizando os ciclos lunares. Através de algoritmos baseados no ciclo sinódico, a aplicação entrega dados precisos sobre o movimento da seiva, janelas de germinação e períodos ideais de colheita.



## ✨ Funcionalidades

* **Cálculo Lunar em Tempo Real:** Algoritmo de precisão baseado no ciclo de 29.5 dias.
* **Dual Hemisphere Support:** Inversão dinâmica da máscara lunar para Hemisfério Sul e Norte.
* **Painel Bio-Sincronizado:** Indicadores de Fluxo de Seiva, Nutrição Ideal e Risco de Estresse.
* **Calendário Interativo:** Navegação por meses e anos com visualização de atividades por ícones.
* **Interface Glassmorphism:** Design futurista com tema *Deep Space Green*, otimizado para dispositivos móveis.
* **Guia Educativo:** Documentação técnica integrada baseada em fundamentos de botânica lunar.

## 🚀 Tecnologias Utilizadas

O projeto foi construído utilizando o "Vanilla Stack" (puro), garantindo performance máxima e zero dependências externas:

* **HTML5** - Estrutura semântica.
* **CSS3** - Estilização avançada com variáveis, Glassmorphism e animações.
* **JavaScript (ES6+)** - Lógica astronômica e manipulação dinâmica do DOM.
* **SVG Dinâmico** - Renderização em tempo real das fases da lua via código.
