# growlendario
O Growlendário 2026 é uma ferramenta premium para cultivadores que utilizam o ciclo lunar. Com interface Glassmorphism e cálculos astronômicos precisos, o app monitora o fluxo de seiva, janelas de germinação e colheita. Desenvolvido por William Santana, une botânica e tecnologia para elevar o nível do seu cultivo orgânico ou mineral. 🌙🌿
